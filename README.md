# House Price Prediction System

## Overview

This project predicts house prices using Machine Learning models and provides predictions through a Streamlit web application.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

## Features

- Data preprocessing
- Model training
- Multiple regression model comparison
- Model evaluation using:
  - R² Score
  - MAE
  - MSE
  - RMSE
- Real-time prediction through Streamlit
